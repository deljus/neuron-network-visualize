# neuron-network-visualize 
:rocket: :rocket: :rocket:

Desktop application for visualizing changes in weights, errors for a fully connected neural network.



## Development

Clone the application and run the following commands

```$xslt
cd %PROJECT_FOLDER%
yarn
yarn run dev
```

## Running the tests

```$xslt
yarn run test
```

## Build

```$xslt
yarn run build
```
